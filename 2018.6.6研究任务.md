6.生成器

目前的生成器比较的简单，只是一个文件拷贝的功能，后续会根据需求改进。

```sh
git clone https://github.com/dafuyi/mpvue-generator.git

cd mpvue-generator && npm i

npm link
```

之后，就全局的使用`gwx`
```
gwx new <page name>
```

>注意：请在`src/pages/`目录下使用
